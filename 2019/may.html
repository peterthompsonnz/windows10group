May: Revisit using the Mail app, and explore the privacy features of the Firefox and Opera web browsers

Last session we looked at how to manage several email accounts using the Mail app. This session we will recap what we learned and then explore the other features of the Firefox and Opera web browsers. We will look at reading mode in Firefox, Opera's VPN feature and privacy options each browser provides. For example the uBlock Origin add-on.
