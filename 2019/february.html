File and Folder Management

Managing files and folders in an efficient manner is one of the core skills necessary to be a competent computer user. This session will look at:

  • The tree-like structure that Windows uses to organise files
  • The naming of files and folders
  • How to organise files into folders and sub-folders
  • How to select, move, copy, delete, and rename files and folders

If time allows we will look at managing multiple windows.
